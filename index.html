Welcome to TeamVR Senior Capstone Project.

Welcome to our CSCE 482 Capstone Project. The project consists of making an interactive, virtual reality system where users can learn and add more content based on their learning needs. The project will try to fill the need for the integration of learning online and computer tools into a more hands-on and traditional learning experience. It will require the use of the Oculus Rift wearable headset and the LeapMotion hand gesture tracking device. We expect that our results will create a better teaching system for schools and a better learning method for users. The objectives include to create a menu system from where users can add more content, create an application to demonstrate the learning possibilities, and make the whole project easy for schools to use.
